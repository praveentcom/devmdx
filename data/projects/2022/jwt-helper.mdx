## What It Does

JWT Helper takes the pain out of working with JSON Web Tokens. Generate tokens with a single function call and validate them just as easily.

## Features

- Simple API with TypeScript support
- Sensible defaults for common use cases
- Works with any Node.js framework
- Comprehensive test coverage

## Tech Stack

Pure TypeScript with zero runtime dependencies. Published on npm and works with Node.js 14+.
