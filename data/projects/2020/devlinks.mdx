## What It Does

DevLinks gives developers a clean, customizable page to share all their important links in one place. Think Linktree but built for tech folks.

## Features

- Custom themes and layouts
- Analytics for click tracking
- Import links from GitHub, Twitter, etc.
- Custom domain support

## Tech Stack

Next.js with TypeScript for the app, PostgreSQL for data, and Tailwind CSS for styling. Deployed on Vercel.
