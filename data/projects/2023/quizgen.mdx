## What It Does

QuizGen uses GPT-4 to generate high-quality multiple choice questions on any subject. Perfect for self-study or creating assessments.

## Features

- Generate questions by topic or difficulty
- Save and organize question sets
- Track your progress over time
- Export to PDF or Markdown

## Tech Stack

Next.js 14 with App Router, TypeScript, OpenAI API for generation, and PostgreSQL with Prisma for storage.
